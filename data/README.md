# data/

사이트가 사용하는 `courses_grouped.json`의 **원본 소스**가 들어가는 폴더입니다.

## 사용 방법

1. 사내 시스템에서 추출한 엑셀을 `data/` 폴더에 저장합니다.
   - 파일명은 자유 (`courses.xlsx`, `courses_v2.xlsx`, `courses_2026-05.xlsx` 등 무엇이든 OK).
   - 빌드 스크립트가 `data/` 안의 **가장 최근에 저장된 .xlsx 파일**을 자동으로 선택합니다.
   - 이전 버전은 보관용으로 그대로 두셔도 되고, 깔끔하게 지우셔도 됩니다.
2. 처음 한 번만 의존성 설치:
   ```
   npm install
   ```
3. 빌드 명령 실행:
   ```
   npm run build:courses
   ```
4. 자동으로 갱신된 `courses_grouped.json`을 커밋 & 푸시합니다.

## 규칙

- 컬럼 **순서는 자유**. 컬럼 **이름(헤더)**으로 읽기 때문에 순서가 매번 달라져도 문제 없습니다.
- 알 수 없는 부가 컬럼은 무시되며, 필수 컬럼이 빠지면 빌드가 어떤 헤더가 빠졌는지 알려줍니다.
- `웹공개 = Y` 인 행만 사이트에 노출됩니다.
- `courses_grouped.json`은 **직접 수정하지 마세요**. 이 엑셀에서 자동 생성됩니다.

## 필수 헤더 (15개)

```
과정명, 대분류, 중분류, 과정유형, 교육구분,
기준단가, 일반회원, 교육일수, 총교육시간,
No, 시작일, 종료일, 지역구분, 주소(URL), 웹공개
```
