# 인재키움 직무교육센터

한국생산성본부(KPC)에서 운영하는 **인재키움 직무교육센터**의 공식 안내 사이트입니다.

🔗 **라이브**: https://injae.vercel.app

## 이 사이트는 무엇인가요?

우선지원대상기업에 다니는 재직자 분들을 위한 직무교육 안내 페이지입니다.
교육을 수료하면 **수강료의 90%를 환급**받을 수 있어, 실제 부담은 정가의 10% 수준입니다 (고용보험기금 환급).

- 어떤 과정을 들을 수 있는지
- 우리 회사가 환급 대상인지
- 어떻게 신청하는지

위 세 가지를 한눈에 확인할 수 있도록 만든 정적 웹사이트입니다.

## 누구에게 도움이 되나요?

- **재직자** — 본인이 들을 수 있는 과정과 비용을 빠르게 확인
- **기업 인사·교육 담당자** — 직원 교육 계획 수립용 과정 카탈로그
- **KPC 운영팀** — 안내 페이지 단일 진입점

## 주요 페이지

| 페이지 | 설명 |
|---|---|
| **`/` (`index.html`)** | 90% 환급 제도 소개, 8개 직무 분야, 신청 4단계, FAQ |
| **`/courses.html`** | 전체 과정 검색·필터 (201개 고유 과정, 575개 차수). 분야·지역·교육 기간으로 필터링하고 과정명을 클릭하면 일정별 차수까지 확인 가능 |

## 기술 구조 (개발자용)

복잡한 빌드 없이 누구나 바로 손볼 수 있도록 만들었습니다.

- **순수 HTML / CSS / Vanilla JS** — 빌드 단계 없음
- **데이터**: `courses_grouped.json` 한 파일에서 동적으로 카드를 그림 (filter/검색/정렬 모두 클라이언트 측)
- **호스팅**: Vercel 정적 호스팅 (자동 배포)

### 폴더 구조

```
.
├── index.html              # 메인 랜딩
├── courses.html            # 전체 과정 검색·필터 페이지
├── styles.css              # 공통 스타일 (반응형 포함)
├── courses_grouped.json    # 과정 데이터 (분야·차수 그룹핑)
└── assets/                 # 로고·이미지·favicon
```

### 로컬에서 띄우기

```bash
python3 -m http.server 4173
# 브라우저에서 http://localhost:4173 접속
```

> ⚠️ `courses.html`이 `fetch()`로 JSON을 읽기 때문에 반드시 정적 서버 경유 필요.
> `file://` 로 직접 열면 CORS 오류로 과정 목록이 비어 있게 됩니다.

### 변경 → 배포 흐름

1. 새 브랜치에서 작업 (예: `fix/...`, `docs/...`)
2. PR 생성 → main에 머지
3. Vercel이 자동으로 빌드 후 https://injae.vercel.app 에 반영 (1~2분)

`main` 브랜치는 직접 push가 막혀 있어 항상 PR을 거칩니다.

### 모바일 반응형 정책

데스크탑(961px 이상)의 디자인은 **건드리지 않은 채** 모바일 전용 미디어 쿼리만 추가하는 방향으로 유지하고 있습니다.

- ≤960px: 햄버거 메뉴, hero 그리드 1열, 필터 collapsible
- ≤720px: hero/타이틀 폰트 축소, nbsp 강제 줄바꿈 허용 등
- ≤480px / ≤360px: 폰 전용 padding·폰트 추가 축소
